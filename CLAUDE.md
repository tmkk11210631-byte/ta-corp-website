# T.A株式会社 コーポレートHP — プロジェクトルール

## 概要
T.A株式会社（製造・小売業）のコーポレートウェブサイト。
チャットボット付きのホームページを構築するプロジェクト。

## コミュニケーション
- **返答は常に日本語で行う**

## デザイン方針
- **モダンで洗練されたデザイン**を維持する
  - カラーパレット: 黒（#0A0A0A）、深紅（#C0392B）、ゴールド（#D4AF37）
  - フォント: Noto Sans JP（Google Fonts）
  - アニメーション・ホバー効果を適切に活用し、高級感を演出する
- **レスポンシブ対応は必須**
  - モバイル（375px〜）、タブレット（768px〜）、デスクトップ（1280px〜）すべてで表示を確認すること
  - Tailwind CSS のブレークポイント（`sm:` / `md:` / `lg:`）を活用する
  - テキストサイズ・余白・カラム数はビューポートに合わせて調整する

## コーディング規約
- **コードのコメントは日本語で書く**
- TypeScript を使用し、型を明示する
- コンポーネントは `components/` ディレクトリに配置する
- API ルートは `app/api/` ディレクトリに配置する

## 技術スタック
- **フレームワーク**: Next.js (App Router)
- **スタイリング**: Tailwind CSS
- **フォーム**: React Hook Form
- **言語**: TypeScript

## ページ構成（1ページスクロール型）
| セクション | ID | 説明 |
|---|---|---|
| ナビゲーション | — | スクロールで背景変化 |
| ヒーロー | `#hero` | キャッチコピー「安心で、頼れるを実現」 |
| 会社概要 | `#about` | 理念・代表メッセージ |
| サービス | `#services` | 提供サービス・製品カード |
| 料金プラン | `#pricing` | 3プラン比較 |
| 実績 | `#results` | 事例3件 |
| お問い合わせ | `#contact` | メール送信フォーム |
| チャットボット | — | サイト内チャット（今後実装） |

## 環境変数
メール送信には `.env.local` に以下を設定すること（`.env.local.example` 参照）：
```
SMTP_HOST=
SMTP_USER=
SMTP_PASS=
CONTACT_EMAIL=
```
